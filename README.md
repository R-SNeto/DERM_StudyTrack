# Study Track 🚀

Este repositório é dedicado a prototipação de modelagem de uma ferramenta de autoria própria, o **StudyTrack**

## 📌 Funcionalidades
- **Flashcards:** Memorização ativa com repetição espaçada.
- **Cronograma:** Organização semanal de matérias.
- **Chatbot:** Assistente virtual para tirar dúvidas.

## 🛠️ Tecnologias
- Modelagem UML
- UI/UX Design (Figma)
