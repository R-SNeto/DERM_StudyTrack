# Study Track 🚀

O **StudTrack** é uma ferramenta desenhada para auxiliar estudantes em sua vida acadêmica

## 📌 Funcionalidades
- **Flashcards:** Memorização ativa com repetição espaçada.
- **Cronograma:** Organização semanal de matérias.
- **Chatbot:** Assistente virtual para tirar dúvidas.

## 🛠️ Tecnologias
- Modelagem UML
- UI/UX Design (Figma)
